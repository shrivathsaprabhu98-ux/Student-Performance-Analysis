# Student-Performance-Analysis
Student Performance Analysis using Excel and SQL

## Project Overview

This project analyzes student performance data using Microsoft Excel and SQL. The objective is to identify the factors affecting academic performance through data cleaning, KPI analysis, Pivot Tables, charts, and SQL queries.

---

## Tools Used

- Microsoft Excel
- SQL

---

## Dataset

The dataset contains information about:

- Student Name
- Age
- Attendance
- Exam Score
- Video Games
- Tuition
- Health
- Stress
- Daily Work
- Self Study

---

## Data Cleaning

- Removed duplicate records
- Filled missing numerical values using the average
- Filled missing categorical values using the mode
- Prepared data for analysis

---

## KPI Summary

- Total Students
- Average Age
- Average Attendance
- Average Exam Score
- Highest Exam Score
- Lowest Exam Score
- Average Self Study Hours

---

## Analysis Performed

- Stress vs Exam Score
- Self Study vs Exam Score
- Attendance vs Exam Score
- Health vs Exam Score
- Tuition vs Exam Score
- Daily Work vs Exam Score

---

## Key Insights

- Students with low stress achieved higher exam scores.
- Higher self-study hours were associated with better academic performance.
- Students with good attendance scored significantly higher.
- Good health positively influenced academic performance.
